Screenshots Directory

This directory is for game screenshots that can be used in the README and documentation.

Suggested screenshots to add:
1. Main menu
2. Gameplay on different board sizes
3. Tournament mode results
4. Game statistics screen

To take a screenshot:
- Windows: Use the Snipping Tool or press Win+Shift+S
- Mac: Press Command+Shift+4
- Linux: Use a tool like Flameshot or press PrintScreen
