Sound Effects for Advanced Tic-Tac-Toe

Place the following .wav files in this directory to enable sound effects:

1. move.wav - Played when a move is made
2. win.wav - Played when a player wins
3. lose.wav - Played when a player loses
4. tie.wav - Played when the game ends in a tie

You can find free sound effects at:
- https://freesound.org/
- https://mixkit.co/free-sound-effects/
- https://www.zapsplat.com/

Note: Sound effects require pygame to be installed:
pip install pygame
